# MISU 2차 패치

## 목적
1차 안정화 이후 남은 실무 기능 보강 패치입니다.

## 포함 내용
- 알토란 엑셀 추출 필터 추가
  - 지역, 부과항목, 최소금액, 최대금액, 3만/5만/30만 빠른버튼
  - 0원 포함/제외, 선납 포함/제외
  - 전화번호 없음 제외, 지로희망자 제외, 문자제외자 제외
  - 엑셀 컬럼: 코드, 상호, 대표자명, 기타사원, 핸드폰, 거래처구분, 품목 코드, 지로발행명목, 규격(월분), 발행연월일, 발행금액
  - 코드 000001 형식

- 제외자/지로희망자 관리 UI 추가
  - 추가/수정/해제 모달
  - 지역, 성명, 차량번호, 관리번호, 핸드폰, 제외유형, 사유, 시작일, 메모
  - 브라우저 localStorage 기반 저장
  - 알토란 추출 필터와 연동

- 통장매칭 2차 보강
  - 수동처리 항목: 회비반영, 가수금, 잡수입, 카드결제, 현금결제, 기타
  - backend 수입 항목에 카드결제/현금결제 추가
  - 매칭결과 초기화 버튼이 /api/deposits/pending DELETE 호출
  - 반영 버튼 optimistic update 일부 적용
  - 글씨/간격 일부 축소

- 폐업/이탈 등록 모달 보강
  - 접수번호 예시: 폐-26 / 양-18 / 이-8 / 탈-3
  - 처리내용/사유 → 메모(선택)
  - 모달 폭 확장

## 적용 파일
- frontend/static/app/Regional.jsx
- frontend/static/app/BankMatching.jsx
- frontend/static/app/Modals.jsx
- frontend/static/app/App.jsx
- backend/app/routers/deposits.py

## 주의
- 제외자/지로희망자 수동 추가 데이터는 현재 브라우저 저장 기반입니다. 다른 PC와 공유하려면 추후 DB 테이블/API가 필요합니다.
- 이번 2차는 알토란/제외자/통장매칭/폐업모달 중심입니다.

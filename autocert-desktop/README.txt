[자격증명 자동인쇄 프로그램]

1. 개요
이 프로그램은 화물운송종사자격증명 template.hwp 파일에 입력한 정보를 자동으로 채워넣고 인쇄하는 도구입니다.

2. 사용 방법
- 프로그램을 실행합니다.
- 관리번호, 성명, 자격증번호 등 필요한 정보를 입력합니다.
- '바로 인쇄' 버튼을 누르면 즉시 기본 프린터로 출력됩니다.
- '한글에서 확인 후 인쇄'를 누르면 한글 프로그램이 열리고 내용을 확인한 뒤 직접 인쇄할 수 있습니다.

3. 주의 사항
- PC에 '한컴오피스 한글'이 반드시 설치되어 있어야 합니다.
- template.hwp 파일이 프로그램 설치 폴더에 존재해야 합니다.

4. 시스템 요구사항
- Windows 10 이상
- .NET Desktop Runtime 8.0 이상
- 한컴오피스 한글
